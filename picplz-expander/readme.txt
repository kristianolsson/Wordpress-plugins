Scans posts for picplz links and expands them to show the image.

== Description ==

This plugin will scan all posts for links to picplz.com/xxx links and add a image tag 
to the end of the post showing the picplz image.

== Installation ==

To install the Picplz Expander:

1. Upload 'picplz-expander.php' and 'picplz-expander.js' to '/wp-content/plugins/picplz-expander/' (create the picplz-expander directory)
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Include picplz links in your posts
